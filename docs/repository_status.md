# Repository status

This repository is the **peer-review access version** associated with the manuscript submitted to *Digital Discovery* (Royal Society of Chemistry).

It is not the final archival release. The structure, file naming, and contents may be revised in response to peer review.

After acceptance, a cleaned and frozen public release will be deposited on Zenodo with a permanent DOI, and that DOI will be cited in the published version of the manuscript.

For this peer-review release, only the embedding store is included under `data/knowledge_base/`. The full curated source documents that compose the SVET knowledge base are retained by the authors and will be addressed in the final archival release after peer review.
